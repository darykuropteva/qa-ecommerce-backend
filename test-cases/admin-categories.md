# Тест-кейсы: Управление категориями

## Описание

Тестирование функциональности управления категориями в административном модуле.

## Покрываемые эндпоинты

* POST /api/categories
* PUT /api/categories/{id}
* DELETE /api/categories/{id}
* GET /api/categories
