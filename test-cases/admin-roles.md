# Тест-кейсы: Роли и доступ

## Описание

Тестирование разграничения прав доступа между ролями ADMIN и USER для административных эндпоинтов системы.

## Проверяемые роли

* ADMIN
* USER

## Проверяемые эндпоинты

* POST /api/categories
* DELETE /api/categories/{id}
* POST /api/products
* Другие административные эндпоинты

## Проверенные сценарии

* ADMIN имеет доступ к административным операциям.
* USER не может создавать категории.
* USER не может создавать товары.
* USER не может удалять категории.
* Система возвращает корректный HTTP-статус 403 Forbidden при недостаточных правах доступа.
